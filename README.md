# words_web_scraping

## Main Goal
 In this small project the goal is to web scraper the main words from job opportunities in Linkedin and show them using wordclound.
 
 ## Requirements
 - Python
 - Selenium webdriver
 - Beautifulsoup
 - Wordcloud
 - Matplotlib
 - Pandas
